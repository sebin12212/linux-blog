# 🐧 Linux 명령어 정리 블로그

## 소개

Linux를 공부하면서 자주 사용하는 명령어들을 한눈에 보기 쉽게 정리한 HTML 블로그 페이지입니다.

---

## 만든 이유

Linux 수업에서 배운 명령어들이 많아지면서 헷갈리는 경우가 생겼고,  
나중에 빠르게 찾아볼 수 있도록 직접 정리해두기 위해 만들었습니다.

---

## 주요 내용

| 섹션 | 내용 |
|------|------|
| 📁 파일 및 디렉토리 | ls, cd, mkdir, rm, cp, mv 등 |
| 📄 파일 내용 확인 | cat, head, tail, grep, wc 등 |
| 🔐 파일 권한 | chmod, chown, 권한 표기법 |
| ⚙️ 프로세스 관리 | ps, kill, top, bg, fg 등 |
| 🌐 네트워크 | ping, ssh, curl, wget 등 |
| ✨ 기타 유용한 명령어 | sudo, alias, tar, pipe 등 |
