📁 파일 및 디렉토리 기본
명령어설명예시
ls파일 목록 출력ls -al
cd디렉토리 이동cd /home
pwd현재 경로 출력pwd
mkdir디렉토리 생성mkdir test
rmdir빈 디렉토리 삭제rmdir test
rm파일/디렉토리 삭제rm -rf test
cp파일 복사cp a.txt b.txt
mv파일 이동 / 이름 변경mv a.txt b.txt
touch빈 파일 생성touch file.txt
find파일 검색find . -name "*.txt"
💡 Tip: ls -al 은 숨김 파일까지 포함한 전체 목록을 자세히 보여줘요.
📄 파일 내용 확인 기본
명령어설명예시
cat파일 전체 내용 출력cat file.txt
head파일 앞부분 출력head -5 file.txt
tail파일 뒷부분 출력tail -5 file.txt
less페이지 단위로 보기less file.txt
grep문자열 검색grep "hello" file.txt
wc줄/단어/글자 수 카운트wc -l file.txt

$ grep "error" log.txt

$ wc -l file.txt

🔐 파일 권한 중요

권한 문자 의미

r read — 읽기 (4)
w write — 쓰기 (2)
x execute — 실행 (1)
- 권한 없음 (0)

chmod 755 분해

rwx 소유자 (7)
r-x 그룹 (5)
r-x others (5)
명령어설명예시
chmod파일 권한 변경chmod 755 file.sh
chown파일 소유자 변경chown user file.txt
chgrp파일 그룹 변경chgrp group file.txt
umask기본 권한 마스크 설정umask 022
⚙️ 프로세스 관리 중요
명령어설명예시
ps실행 중인 프로세스 확인ps aux
top실시간 프로세스 모니터링top
kill프로세스 종료kill -9 1234
killall이름으로 프로세스 종료killall firefox
bg프로세스 백그라운드 실행bg %1
fg프로세스 포그라운드로 전환fg %1
jobs백그라운드 작업 목록jobs
💡 Tip: kill -9 는 강제 종료예요. 일반 종료는 kill -15 (SIGTERM) 를 먼저 시도해보세요.

✨ 기타 유용한 명령어 유틸
명령어설명예시
man명령어 매뉴얼 확인man ls

clear터미널 화면 지우기clear
